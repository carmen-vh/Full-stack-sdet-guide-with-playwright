# Full-stack-sdet-guide-with-playwright

This repository contains my learning notes, exercises, and test automation work
from a Playwright course. The goal of this project is to develop a strong
practical understanding of browser automation, UI testing, and modern test
workflows using Playwright.

---

## Tech Stack

- **Playwright**
- Node.js
- JavaScript / TypeScript *(depending on project setup)*
- VSCode

---

## What I've Learnt
  
✔ Running tests across Chromium, Firefox, and WebKit  
✔ Locators and selectors  
✔ Assertions and waiting strategies 
✔ Visual and device emulation testing  
✔ Page Object Model (POM) design pattern  
✔ Parallel execution & test organization  
✔ API Testing
